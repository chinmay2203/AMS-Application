# AMS Desktop Application

## Description
This application wraps the AMS web system into a desktop interface using Python.
When executed, it launches a desktop window and loads the AMS web application.

## Technologies Used
- Python
- Flask
- PyWebView (Desktop Window Wrapper)

## How to Run

1. Install Python 3.10 or higher.

2. Install required dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py

The desktop window will open and load the AMS system.

## Project Structure

- app.py              → Main application file
- templates/          → HTML templates
- uploads/            → Uploaded files (if applicable)
- requirements.txt    → Python dependencies

## Notes
- This project is intended as a desktop wrapper for the existing AMS web system.
- No external browser is required.
